## HTML Resume Assignment
This project is a simple resume webpage created using only HTML as part of full stack development course assignment.

## Technologies Used
- HTML5

## Objectives
- To understand the structure of an HTML document
- To practice basic HTML tags such as headings, paragraphs, lists, tables, and links

## Note
- This project does not include CSS or JavaScript as per assignment requirements

## Future Scope
- Add CSS for styling
- Make the resume responsive
