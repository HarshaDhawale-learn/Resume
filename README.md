## HTML Resume Assignment
This project is a simple resume webpage created using HTML and CSS as part of full stack development course assignment.

## Technologies Used
- HTML5
- CSS (added for basic styling)

## Objectives
- To understand the structure of an HTML document
- To practice basic HTML tags such as headings, paragraphs, lists, tables, and links
- To understand how HTML and CSS work together

## Future Scope
- Make the resume responsive
